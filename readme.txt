This repository includes the javascript code of the personal analytics dashboard produced by the [UCIAD](http://uciad.info) project. It makes use of the sparql.js library by Lee Feigenbaum, Elias Torres and Wing Yung. It assumes that a SPARQL 1.1 endpoint is deployed at http://localhost:3030/uciad/query.
